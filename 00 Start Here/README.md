# Learning Atlas

这是合并后的主学习入口。先从这里判断你要学习哪一类内容，再进入对应文件夹。

## Recommended Paths

- `01 Product/`：产品经理、消费级产品、订阅、推送、动线、埋点。
- `02 AI PM/`：AI 产品经理知识体系、AI 时间线、Agent、RAG、AI 工具和面试。
- `03 Coding/`：Vibe Coding、Git、前后端基础、数据库、部署、调试与安全。
- `04 Website/`：个人网站、网页基础、内容结构、设计基础和上线流程。
- `05 Career/`：职业资料、沟通表达、求职、行业报告。
- `06 Courses/`：英语、VS Code、Web 基础练习等成套课程。
- `07 Reference Library/`：大型 PDF、PPT、Word、Axure、模板和资料库。
- `08 Wellbeing/`：增肌、生理学和健康学习资料。
- `09 Personal Atlas/`：个人画像、记忆、对话上下文和自我理解材料。
- `_Review Inbox/`：旧版本来源、待复核材料和不适合直接并入主线的内容。

## How To Use

优先读 `01-05` 的轻量 Markdown 学习系统；需要模板、PDF、PPT、Axure 或大资料时，再去 `07 Reference Library/`。
旧版来源不会覆盖主版本，统一放在 `_Review Inbox/`，方便以后慢慢筛。
