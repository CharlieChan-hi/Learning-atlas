## How To Use Evidence

Use sources in three layers:

```text
Layer 1: Official rules
  Apple / Google
  Use for compliance and required wording.

Layer 2: Industry reports
  RevenueCat / Superwall / Appcues / Amplitude / NNG
  Use for benchmarks and common patterns.

Layer 3: Social discussions
  X / HN / operator posts
  Use for judgment, examples, and current debates.
```
