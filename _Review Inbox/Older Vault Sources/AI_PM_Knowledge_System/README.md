## 📐 排版说明

本资料库的排版遵循以下原则：

**杂志感设计**
- Emoji 图标分区，视觉层次清晰
- 大量留白，段落简短（3—5 行）
- 表格辅助对比，引用框高亮重点

**中英双语**
- 正文中文，核心段落后附英文精简版
- AI 术语首次出现标注英文原名
- 每节末尾有英文要点总结

**三端兼容**
- Obsidian：日常阅读与笔记
- Notion：导入分享
- 纸质打印（A4）：最佳阅读体验

> 💡 **提示：** 如果你打印阅读，建议用 A4 纸、单面打印。
> 每篇文件控制在 8—12 页以内，适合单次阅读。
