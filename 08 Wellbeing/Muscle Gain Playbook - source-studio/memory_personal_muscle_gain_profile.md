## 五、饮食偏好与执行偏好

| 项目 | 内容 |
| --- | --- |
| 不喜欢 | 花生酱、内脏 |
| 可接受食物 | 鸡蛋、鸡腿、鸡胸、牛肉、豆腐、豆浆、酸奶、奶粉、燕麦、香蕉 |
| 喜欢主食 | 面包、包子 |
| 现有午餐习惯 | 常吃楼下沙县鸡腿饭 |
| 外卖风险 | 晚上下班后容易路边随便买垃圾食品 |
| 备餐接受度 | 可以周末备餐 |
| 做饭时间接受度 | 20 分钟左右 |
| 做饭条件 | 蒸锅、电饭煲、冰箱、微波炉都有 |

### 已确认更合理的做法

- 鸡蛋不需要一次性煮很多
- 鸡蛋适合常备，现蒸、现炒、现做
- 可以做炒饭，没必要把晚饭限制成全蒸锅路线
- 真正需要提前准备的是鸡肉、米饭 / 隔夜饭、蔬菜和加餐

### 执行偏好

- 接受固定菜单反复吃
- 更适合低复杂度、高重复的方案
- 最容易失败的点是：嫌麻烦、睡太晚、下班后乱吃

- 主训练时间：晚上
- 训练频率：每周 3 次
- 训练结构：A / B 轮换
- 单次时长：50-60 分钟
- 训练重点：基础器械增肌 + 肩颈友好热身

---

## 九、默认饮食策略

- 早餐：主食 + 蛋白 + 饮品
- 午餐：鸡腿饭 / 鸡胸饭 / 牛肉饭类
- 下午：固定加餐
- 晚餐：训练后主餐
- 睡前：视当天摄入补少量蛋白

### 晚餐允许的主力形式

1. 米饭 + 鸡肉 + 豆腐 + 熟蔬菜
2. 蛋鸡肉炒饭 + 一份豆腐或蒸蛋

## 十、必须避免的错误建议

后续如果继续调整计划，下面这些方向不要再当默认建议：

1. 不要再把“一次煮很多鸡蛋”当成长期主建议
2. 不要把“蒸玉米 + 水煮蛋”当作核心增肌餐
3. 不要把晚饭限制成只有蒸锅路线
4. 不要把中午训练当主训练时间
5. 不要给过于复杂、需要高意志力维持的饮食计划
